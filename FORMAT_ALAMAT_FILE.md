# Format Pengalamatan File Project

## 1) Format Absolut (Windows)
`C:\Users\AZHAM\OneDrive\Documents\Joki Azm\IOT\Iot Rendra\<nama-file>`

Contoh:
- `C:\Users\AZHAM\OneDrive\Documents\Joki Azm\IOT\Iot Rendra\index.html`
- `C:\Users\AZHAM\OneDrive\Documents\Joki Azm\IOT\Iot Rendra\script.js`

## 2) Format Relatif (dari root project)
`./<nama-file>`

Contoh:
- `./index.html`
- `./style.css`
- `./script.js`
- `./server.js`
- `./supabase-schema.sql`
- `./.env`
- `./.env.example`
- `./package.json`
- `./.gitignore`
- `./FORMAT_ALAMAT_FILE.md`

## 3) Format URL Lokal (saat dijalankan lewat server)
`http://localhost:3000/<nama-file>`

Contoh:
- `http://localhost:3000/`
- `http://localhost:3000/env.js`
